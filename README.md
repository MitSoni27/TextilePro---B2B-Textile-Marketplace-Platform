# TextilePro---B2B-Textile-Marketplace-Platform
A modern, enterprise-grade B2B textile marketplace connecting manufacturers, wholesalers, retailers, and exporters. Features dual theme (Light/Dark), responsive design, product showcase, quote cart, authentication system, and more. Built with pure HTML, CSS, JavaScript.

 # TextilePro - B2B Textile Business Platform

A modern, fully responsive B2B textile marketplace platform connecting manufacturers, wholesalers, and retailers.

## Live Demo
https://MitSoni27.github.io/TextilePro-B2B-Platform

## Features
- 🌓 Light & Dark Theme Toggle
- 📱 Fully Responsive (Mobile, Tablet, Desktop)
- 🔐 Login/Register Modal System
- 🛒 Quote Cart Functionality
- 🎠 Hero Image Slider
- 📦 Product Showcase with Pricing
- 🏷️ Category Sections
- 📍 Contact & Location Map
- 📧 Newsletter Subscription
- ✨ Smooth Animations

## Tech Stack
- HTML5
- CSS3 (with CSS Variables)
- JavaScript (ES6+)
- Swiper.js for slider
- Font Awesome 6 for icons
